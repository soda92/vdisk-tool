# vdisk-tool

A tool for managing virtual disks.

## 安装

Windows:

安装sudo: `scoop install sudo`

打开管理员权限命令行：`sudo pwsh`

```
.\vdisk-tool install
.\vdisk-tool start
```

打开[localhost:5002](http://localhost:5002)即可查看。
